GuildZip reformate les messages du chat de guilde Hypixel — bridge, normaux et officer — en lignes compactes et épurées. Dépose-le dans ton dossier mods. Ça fonctionne instantanément.

**Bridge** — `Guild > [MVP++] BotName: G > DiscordUser: hey` → `G > Bridge > DiscordUser : hey`

**Avec tag de version** — `... [V2] G > DiscordUser: gg` → `G > V2 > DiscordUser : gg`

**Message normal** *(mode global)* — `Guild > [MVP++] PlayerName: bonjour` → `G > PlayerName : bonjour`

---

## ✨ Fonctionnalités

- 🤖 **Détection automatique** du bot bridge — aucune config requise
- 🏷️ **Labels de version** — V1 / V2 / V3 chacun avec sa propre couleur
- 🎨 **Alias & couleurs** personnalisables — renomme le bot, stylise chaque élément
- 🌐 **Mode global** — formate tous les messages de guilde, pas seulement le bridge
- 🔁 **Compactage des doublons** — messages répétés groupés en `(x2)`, `(x3)`...
- 🔄 **Auto-updater** — notifie et télécharge les nouvelles versions automatiquement
- 🌍 **FR / EN** — entièrement bilingue

---

## 📦 Prérequis

- Minecraft 1.21+ · Fabric Loader 0.18+ · Fabric API
- ModMenu + Cloth Config *(optionnel — pour l'écran de configuration graphique)*

---

## ⚙️ Commandes

```
/gz           Ouvre la config
/gz update    Vérifie les mises à jour
```
